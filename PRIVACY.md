# Privacy

## Principles
- Local-first: data lagres i nettleserens lokale database (IndexedDB).
- Ingen data sendes til server som standard.
- Eksport/import (JSON) støttes for backup.

## Optional features
Hvis sky-synk noen gang legges til:
- Skal være opt-in
- Skal bruke minst mulig data
- Skal være tydelig dokumentert
