# Privacy

## What This App Stores

- Daily logs: energy (1-5), sleep status/hours, optional notes
- Weekly check-ins: weight, optional notes, one adjustment
- Workout logs: type, date/time, optional duration/notes
- Optional reminder preferences in local storage

## Where Data Is Stored

- By default, data is stored locally in browser IndexedDB.
- Core app usage does not require server-side storage.

## Data Sharing

- No automatic cloud sync is enabled in the MVP.
- Backup/export is user-initiated JSON.
- Import is user-initiated and local.

## User Control

- Users can export their data from Settings.
- Users can re-import backups.
- Users can clear browser storage manually through browser settings.

## Notification Permissions

- Browser notifications are opt-in.
- Notification reminders are local and do not send data externally.

## Future Changes

If online sync or integrations are introduced in future versions:

1. They must be opt-in.
2. Data collection must be minimal and documented.
3. Privacy documentation must be updated before release.
